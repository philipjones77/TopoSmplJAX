# Asset Boundary

This repository distinguishes between repository-authored software and third-party body-model assets.

## Repository-Authored Material

Code, tests, examples, tools, contracts, and authored documentation in this repository are MIT-licensed unless a narrower notice says otherwise.

## SMPL And SMPL-X Assets

SMPL-family model files are usually obtained from their upstream distributors under separate terms.

This repository does not grant a license to:

- SMPL model files
- SMPL+H model files
- SMPL-X model files
- scans, textures, or other vendor-distributed body assets

If such files are placed under `private_data/smpl/`, they remain governed by their upstream license or EULA.

## Maintainer Rule

Only check in SMPL-family assets when redistribution is clearly permitted. Otherwise keep the repository limited to:

- conversion tools
- validation tools
- small synthetic fixtures
- metadata and documentation
