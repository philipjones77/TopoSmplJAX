# Data

Keep only small checked-in reference assets and curated lightweight inputs here.

Licensing boundary:

- repository-authored metadata, tiny synthetic fixtures, and helper manifests may be MIT-licensed
- third-party SMPL-family assets placed here are not relicensed by this repository
- upstream license terms continue to govern any imported model files

Local model asset convention:
- `private_data/smpl/models/validated/`: validated canonical model files that should be used by tools/examples
- `private_data/smpl/models/quarantine/`: unvalidated or manual-drop assets that should not be treated as canonical
