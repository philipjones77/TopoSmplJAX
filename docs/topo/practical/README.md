# Practical

Place user-facing practical usage material here.

Use this folder for:

- how-to guides
- operator playbooks
- task-oriented recipes
- workflow walkthroughs
- troubleshooting guides grounded in actual usage

Practical documents explain how to use the system effectively. They are distinct from:

- `theory/`, which explains why the methods work
- `implementation/`, which explains how the code is built internally
- `reports/`, which records measured or official outcomes
