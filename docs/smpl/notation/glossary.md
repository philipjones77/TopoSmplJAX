# Glossary (Version 1.0.0)

- `LBS`: Linear Blend Skinning.
- `JIT`: Just-In-Time compilation in JAX.
- `AD`: Automatic Differentiation.
- `pose2rot=True`: axis-angle pose input path.
- `pose2rot=False`: rotation-matrix pose input path.
- `compile cache`: runtime cache of jitted functions keyed by mode/shape policy.
- `batch bucket`: padded batch size used to stabilize compile keys.
