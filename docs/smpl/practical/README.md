# Practical

Place user-facing practical usage material here.

Use this folder for:

- how-to guides
- runtime recipes
- model preparation workflows
- export and integration walkthroughs
- troubleshooting and operator notes

Practical documents are usage-oriented. They are distinct from:

- `theory/`, which explains underlying concepts
- `implementation/`, which explains engineering internals
- `reports/`, which records measured or official outcomes
